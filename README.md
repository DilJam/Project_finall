# API RESTful para la Gestión de Eventos

Esta es una API RESTful para la gestión de eventos, desarrollada con Node.js y Express. Permite realizar operaciones CRUD sobre eventos almacenados en una base de datos MongoDB.

## Requisitos

- Node.js
- MongoDB Atlas
- GitHub (para el despliegue)(con poco éxito)

## Instalación

1. Clona el repositorio:
   ```bash
   git clone https://github.com/tu_usuario/proyecto_final.git
   cd proyecto_final/backend
   ```

## Despliegue

Para desplegar la aplicación en Heroku utilizando GitHub Actions, sigue estos pasos:

1. Asegúrate de que tu código esté en la rama `main`.
2. Realiza un `push` de tus cambios a la rama `main` en GitHub.
3. La acción de GitHub se activará automáticamente y comenzará el proceso de despliegue.
4. Monitorea la pestaña de "Actions" en tu repositorio de GitHub para verificar el estado del despliegue.

Asegúrate de que la clave API de Heroku esté configurada en los secretos de tu repositorio de GitHub como `HEROKU_API_KEY`.
